# Sillage UI — 验收清单

> 改动 UI 后对照勾选；全部为**可观察 / 可验证**项。令牌正确性以 [`design-system.md`](./design-system.md) 为准，方向以 [`README.md`](./README.md) 为准。

## 通用门槛（每次 UI 改动都必须过）

- [ ] `pnpm --dir web typecheck` 通过
- [ ] `pnpm --dir web lint` 通过（Biome）
- [ ] `pnpm --dir web test` 通过（改了交互逻辑要补 / 更新测试）
- [ ] `pnpm --dir web build` 通过
- [ ] `go test ./...` 全绿（未因样式改动引入逻辑回归）
- [ ] 改动的每一行都能追溯到设计方向（无顺手「改进」无关代码）
- [ ] 新增的每个颜色 / 表面都同时给了 `dark:` 值

## 配色与字体

- [ ] 品牌、导航、选中态与常规反馈为中性灰（无 celadon / clay 等彩色强调残留）
- [ ] 红色只用于错误、删除 / 退出等破坏性语义，不作为品牌强调色、普通选中态或装饰色
- [ ] 强调（主按钮、选中实底）为近黑（浅色）/ 近白（深色）
- [ ] 全站只用无衬线字体，无 serif / 宋体
- [ ] 无残留 `#fff` / `#000` 颜色硬编码；`gray-400` 不承载正文

## 外壳与导航

- [ ] 桌面左侧栏常驻（`18rem`）且可折叠；折叠后主区让位正确、展开按钮可用
- [ ] 主导航为 记录 / 历史；问答经「新问答」+「对话」列表进入；设置在用户菜单
- [ ] 移动端顶部条 + 汉堡 → 抽屉（`18rem`、最大 `88vw`）；点遮罩 / 按 Esc / 路由跳转均能关闭
- [ ] 移动抽屉打开后锁定页面滚动并圈定 Tab 焦点；关闭后焦点回到汉堡按钮
- [ ] `QuickCapture`（⌘/Ctrl+J）在非 `/ask` 页可用；进入 `/ask` 只隐藏入口，返回后全局草稿仍在
- [ ] 侧栏底部快捷主题按钮与「设置 → 外观」入口均可用；任一入口切换后，两处图标、标签、localStorage 与页面主题同步
- [ ] 历史列表可在「当前记录 / 已归档」间切换；搜索请求携带对应状态，并能找到尚未加载到当前列表的服务端记录

## 组件与一致性

- [ ] 复用 `ui.ts` 令牌；新增高频模式已提为令牌并登记到 `design-system.md`
- [ ] 图标操作、视图 / 状态切换与加载占位分别复用 `iconButtonClass`、`segmented*`、`skeletonClass`
- [ ] 长正文复用 typography `prose`（未手写排版），暗色 `dark:prose-invert` 正常
- [ ] 图标统一用 `lucide-react`，配 `aria-label`
- [ ] 新建与编辑草稿按记录隔离：重载可恢复、站内跳转与关闭 / 刷新均有保护、保存后清除对应草稿
- [ ] 编辑草稿的 `baseVersion` 与服务器版本不同时不自动覆盖且不能提交；「保留最新内容 / 恢复我的草稿」两条路径均明确且恢复后仍需保存
- [ ] 编辑器点「取消」时 dirty 草稿需确认；「放弃修改」清除对应草稿，「继续编辑」保留正文和焦点
- [ ] 详情编辑与版本动作等待最新详情快照；正文和 `expectedVersion` 属于同一版本，非 404 错误保留缓存并可重试
- [ ] 从首页、历史筛选、日历和问答来源打开详情后，返回路径保留原查询参数与来源
- [ ] 附件上传时保存与取消不可用，站内跳转先确认；上传 Markdown 插入完成后才恢复保存
- [ ] AI 设置以最近一次读取 / 保存成功的服务端内容为基线，未保存修改同时保护站内跳转、关闭 / 刷新和手动退出登录
- [ ] AI 设置保存 / 删除进行中锁定可修改表单；已有档案删除需二次确认，存在其他 dirty 修改时先拒绝删除
- [ ] 问答发送 / 重新生成与速记保存均为单飞；切换会话会作废待完成的新会话创建，速记只在成功后清除全局草稿
- [ ] 列表分页同一游标只发一次请求；刷新或 canonical 记录变更会作废并重取旧列表 / 页结果
- [ ] 历史列表的置顶组先于日期组；日历补齐全部分页后才显示结果，加载有数量进度，失败只在用户点击后重试

## 无障碍与响应式

- [ ] 焦点环全程可见（`focus-visible:ring-gray-400/40` 一致）；Tab 可达侧栏 / 抽屉 / 浮层
- [ ] 浮层 / 抽屉 / 菜单可 Esc 关闭；模态浮层有 `role="dialog"` / `aria-modal` 与焦点管理，Esc 只关闭当前最上层模态
- [ ] 未保存导航使用 `role="alertdialog"`；Esc / 遮罩选择留在当前页，明确点击「离开此页」才继续原导航
- [ ] 关键文字对比 ≥ AA
- [ ] 窄屏不溢出；主要触控目标 ≥ 40px，浮动速记按钮避让底部 safe area
- [ ] 按钮、链接和 `summary` 使用 `touch-action: manipulation`，触控不会引入无意义的双击缩放延迟

- [ ] 涉及产品形态 / 导航 / 命名的改动，已同步 [`../product-guidance.md`](../product-guidance.md)
- [ ] 无 `console.log` / 调试代码；无硬编码密钥
